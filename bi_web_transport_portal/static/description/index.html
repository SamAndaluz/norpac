Website Freight Transport Management and Delivery Routes Odoo

Portal Delivery Transport Mangement with Freight Odoo Apps

Website Portal for Transport Delivery and Picking Order Odoo Apps

This Odoo apps helps to manage Freight Transportation Management and Logistics System with website portal feature which has key features which include transporter details,Vehicle information, route info, LR number of parcel all delivery order details. This create auto entry transport entry with reporting those entries make it more easy to operate Transport details in Odoo app. In Odoo managing Trans-shipment and vehicle for transporter become super easier. Also this apps provide features to show transport details and status of transport delivery on website portal along with transporter and vehicle information.

Features

User can See Transport Details.

User can see and add transport details in app which include all the details.

Transport Route Details.

User can see and add route details, for better record.

Transporter on Vehicle.

User can select transporter on vehicle so delivery done using this vehicler.

Add Transport Delivery Charge.

User also can add transport delivery change to transporter.

Transporter on Sales Order

User can add transporter on sales order and delivery done based on selected transporter.

Transport Entry Report.

User can print transport entry detail report wich includes route and all transport related information.

Visible Transport Entry in Customer Portal

Customer can see all transport entry details in portal.

Download and Print Transport Entry Report on Portal

Customer can print or download transport entry detail report in pdf format on portal.

Configuration for Portal User

When you create user at that time select portal user type.

Transporters Details Configuration

User can add transporters detail and "Transport Charges" of transporter.

Transporters Vehicle

User can see available vehicle for transport.

Transporters Routes Location Details

Transporters Routes Details

User can set the details of the transporters routes with Source and Destination Location.

Sales Order

While creating new quotation user can assign transporters for sales order delivery.

Delivery Order

On confirming sale order user can see "Tracking Number" of delivery order, user can also enter "LR Number".

By pressing "Enter No of Parcel" button user can add number of parcel of delivery order.

User can see enterd number of parcel added to delivery order.

Transport Route Locations Entry

On validating delivery order user can see generated "Transporters Routes Location Entry".

User can also see related transport entry for the picking.

Start Transport Entry

User can start transport entry by clicking "Start" button on transport entry, state will moved to "In-Progress".

Transport Details of Customer

Customer can see transport details on my account web page in customer portal.

Customer can see all transport entry in transport details web page with sorting by, filter by, group by and search functionality.

Transport Entry Web Page of Customer

Customer can see transport entry details web page with start state in customer portal.

In-Progress Transport Entry

User can done transport entry by clicking "Done" button on transport entry and transport entry will moved to "Done" state.


Customer can see transport entry details web page with In-Progress state in customer portal.

Done Transport Entry


Customer can see transport entry details web page with Done state in customer portal and also download transport entry details in pdf format.

User can download or print the transport entry details report as shown below.
